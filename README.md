Customer Churn Prediction

A Machine Learning project that predicts whether a customer is likely to leave a company’s service or not. This project helps businesses improve customer retention by analyzing customer behavior and identifying potential churn customers.

Features
Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Machine Learning model training
Customer churn prediction system
Flask web application
SQL database setup
Model saving using Pickle
Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn
Flask
SQL
Project Structure
app.py → Flask web application
churn_analysis.ipynb → Data analysis and model training
predict_customerchurn.ipynb → Prediction notebook
models/churn_model.pkl → Trained ML model
database/setup.sql → Database setup file
Churn_Data.csv → Dataset
requirements.txt → Required libraries
Machine Learning Algorithms
Logistic Regression
Decision Tree
Random Forest
Objective

The main goal of this project is to help companies reduce customer loss by predicting churn in advance and taking preventive actions.
